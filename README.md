# Supply Chain Analytics – Correlation Matrix Visualization

**Email:** 24f2004876@ds.study.iitm.ac.in

This repository contains the required files for the Supply Chain Correlation Matrix assignment:

## Files Included
1. **correlation.csv**  
   - Contains the correlation matrix generated using Excel’s Analysis ToolPak.

2. **heatmap.png**  
   - Screenshot of the Excel correlation heatmap using the Red–White–Green color scale  
     (Red = low correlation, White = medium, Green = high correlation).

## Notes
- Correlation matrix created by selecting all five variables with "Labels in first row" option.
- Conditional formatting applied using: **Home → Conditional Formatting → Color Scales → Red–White–Green**.
- The heatmap image is within the required resolution range (400×400 to 512×512 px).
